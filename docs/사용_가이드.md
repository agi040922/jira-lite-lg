## ✨ 구현된 주요 기능

### 1. 대시보드 (`/dashboard`)
- 내가 할당된 이슈 목록
- 최근 활동 내역
- 팀 통계 및 인사이트
- AI 기반 통계 분석

### 2. 프로젝트 관리 (`/projects`)
**할 수 있는 것:**
- ✅ 새 프로젝트 생성
- ✅ 프로젝트별 칸반 보드 (Drag & Drop)
- ✅ 프로젝트 상태 컬럼 커스터마이징 (Backlog, In Progress, Done 등)
- ✅ 프로젝트 삭제 (Soft Delete - 휴지통으로 이동)

**테스트 시나리오:**
1. "새 프로젝트" 버튼 클릭
2. 프로젝트 이름 입력 (예: "웹사이트 리뉴얼")
3. 프로젝트 키 입력 (예: "WEB") - 이슈 번호에 사용됨 (WEB-1, WEB-2...)
4. 생성된 프로젝트의 칸반 보드에서 이슈 드래그 앤 드롭 테스트

### 3. 이슈 관리 (`/issues`)
**할 수 있는 것:**
- ✅ 이슈 생성 (제목, 설명, 우선순위, 담당자, 라벨)
- ✅ 이슈 상세 보기 및 수정
- ✅ Rich Text Editor로 이슈 설명 작성 (이미지 삽입 가능)
- ✅ 서브태스크 추가
- ✅ 댓글 작성
- ✅ 이슈 히스토리 자동 기록
- ✅ 담당자 변경 시 자동 알림

**테스트 시나리오:**
1. "이슈 만들기" 버튼 클릭
2. 이슈 정보 입력:
   - 제목: "로그인 페이지 디자인 개선"
   - 우선순위: High
   - 라벨: UI/UX
3. 설명란에 이미지 붙여넣기 테스트
4. 생성 후 칸반 보드에서 드래그로 상태 변경

### 4. 팀 관리 (`/team`)
**할 수 있는 것:**
- ✅ 팀 생성
- ✅ 팀원 초대 (이메일로 초대 링크 발송)
- ✅ 역할 관리 (ADMIN / MEMBER)
- ✅ 팀 통계 확인

**테스트 시나리오:**
1. "새 팀 만들기" 버튼 클릭
2. 팀 이름 입력
3. "팀원 초대" → 이메일 주소 입력
4. 초대 이메일이 발송되고, 링크로 접속하면 팀 가입

### 5. 칸반 보드 (Drag & Drop)
**할 수 있는 것:**
- ✅ 이슈를 드래그해서 상태 변경
- ✅ 상태 컬럼 추가/수정/삭제
- ✅ 실시간 동기화 (여러 사용자가 동시에 작업 가능)

**테스트 시나리오:**
1. 프로젝트 진입
2. 이슈 카드를 "Backlog"에서 "In Progress"로 드래그
3. 브라우저 2개 창을 열어서 실시간 동기화 확인

### 6. AI 기능
**할 수 있는 것:**
- ✅ 이슈 자동 요약
- ✅ 이슈 설명 AI 생성
- ✅ 댓글 자동 제안
- ✅ 대시보드 인사이트 분석

**테스트 시나리오:**
1. 이슈 상세 페이지에서 "AI 요약" 버튼 클릭
2. 이슈 설명이 자동으로 요약됨
3. 대시보드에서 AI 기반 통계 확인

### 7. 알림 및 인박스 (`/inbox`)
**할 수 있는 것:**
- ✅ 담당자 변경 시 자동 알림
- ✅ 댓글 작성 시 관련 사용자에게 알림
- ✅ 알림 읽음 처리

### 8. 인사이트 (`/insights`)
**할 수 있는 것:**
- ✅ 팀 생산성 통계
- ✅ 이슈 완료율 그래프
- ✅ 멤버별 작업량 분석

### 9. 휴지통 (`/trash`)
**할 수 있는 것:**
- ✅ 삭제된 프로젝트 복구
- ✅ 영구 삭제

## 🎯 빠른 체험 순서

### 첫 방문 시 (5분 코스)
1. Google 계정으로 로그인
2. 대시보드에서 전체 현황 확인
3. "새 프로젝트" 생성
4. 프로젝트 안에서 이슈 1~2개 만들기
5. 칸반 보드에서 드래그 앤 드롭 테스트

### 심화 테스트 (10분 코스)
1. 팀 생성 및 팀원 초대
2. 이슈에 이미지 첨부 및 Rich Text 작성
3. 서브태스크 추가
4. 담당자 변경 후 알림 확인
5. AI 요약 기능 테스트
6. 프로젝트 삭제 → 휴지통에서 복구

## 🛠️ 구현된 기술 스택

- **Frontend**: Next.js 15 (App Router), React 19, TypeScript
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Realtime)
- **AI**: Anthropic Claude API
- **Email**: Resend API
- **Styling**: Tailwind CSS 4
- **Editor**: React Quill (Rich Text Editor)

---

## 📌 주요 특징

### 1. 실시간 협업
- Supabase Realtime을 활용한 실시간 데이터 동기화
- 여러 사용자가 동시에 칸반 보드 수정 가능

### 2. AI-Native 기능
- 이슈 자동 요약 및 제안
- 대시보드 인사이트 분석
- 댓글 자동 생성

### 3. 이메일 발송
- 팀 초대 이메일 자동 발송
- 비밀번호 재설정 링크 발송

### 4. Soft Delete
- 프로젝트 삭제 시 휴지통으로 이동
- 30일 이내 복구 가능

### 5. 자동화
- 이슈 키 자동 생성 (프로젝트-번호 형식)
- 프로젝트 생성 시 기본 상태 3개 자동 생성
- 이슈 변경 이력 자동 기록
- 담당자 변경 시 자동 알림

## 💡 사용 팁

1. **Google 로그인 권장**: 별도 회원가입 없이 바로 시작 가능
2. **프로젝트 키는 짧게**: 이슈 번호에 사용되므로 2~4글자 추천 (예: WEB, API, UI)
3. **이미지 붙여넣기**: 에디터에 Ctrl+V로 이미지 바로 삽입 가능
4. **실시간 동기화 테스트**: 브라우저 2개 창에서 동시 작업해보세요
5. **AI 기능 활용**: 복잡한 이슈는 AI 요약으로 핵심 파악

## ❓ 자주 묻는 질문

**Q: 회원가입 없이 둘러볼 수 있나요?**
A: 테스트 계정 (admin@test.com / Test1234!)으로 바로 로그인하세요.

**Q: 팀원을 어떻게 초대하나요?**
A: 팀 관리 페이지에서 이메일 주소를 입력하면 초대 링크가 발송됩니다.

**Q: 삭제한 프로젝트를 복구할 수 있나요?**
A: 네, 휴지통(`/trash`)에서 30일 이내 복구 가능합니다.

**Q: 동시에 여러 사람이 작업할 수 있나요?**
A: 네, Realtime 기능으로 실시간 동기화됩니다.
